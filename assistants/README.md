# Assistants

Conversational AI helpers that respond to user requests. Assistants are reactive agents that work through dialogue, executing tasks based on user input.

## Characteristics

- **Reactive**: Responds to user messages
- **Conversational**: Works through chat interface
- **Task-oriented**: Executes specific tasks on demand
- **Configurable**: Supports multiple modes (chat/task)

## Installation

```bash
yao agent add <owner>/<name>
```

## Directory Structure

```
assistants/
└── <owner>/              # Organization or user
    ├── README.md         # Organization introduction
    └── <name>/           # Individual assistant
        ├── README.md     # Assistant documentation
        ├── meta.json     # Metadata
        └── cover.jpg     # Cover image
```
